> **Interactive web applications and data products with R**\
> *Johns Hopkins Data Science Specialization - Course 9*

## 🎯 Course Overview

This course focuses on creating interactive data products that can be shared with others. Students learn to build web applications, interactive visualizations, and presentations using modern R packages and web technologies. The course emphasizes practical skills for deploying data science solutions in real-world environments.

## 📁 Project Portfolio

### 🗺️ [Leaflet Interactive Maps](leaflet/)

**Technology**: R Leaflet\
**Objective**: Create interactive web maps with custom markers and popups

-   **Project**: London Hospitals Interactive Map
-   **Features**: Clustered markers, responsive design, detailed popups
-   **Skills**: Geographic visualization, HTML widgets, web deployment
-   **Output**: Standalone HTML map application

### 📊 [Plotly Interactive Presentation](plotly-presentation/)

**Technology**: R Plotly + R Markdown\
**Objective**: Build interactive presentations with dynamic visualizations

-   **Features**: Hover interactions, animations, responsive charts
-   **Skills**: Interactive storytelling, web-ready presentations
-   **Output**: Professional HTML presentation
-   **Deployment**: Web-accessible slideshow

### 🎬 [Shiny Web Application](shinyapp/)

**Technology**: R Shiny Framework\
**Objective**: Full-stack web application development

-   **Project**: Movies Data Explorer
-   **Features**: Multiple visualization types, statistical modeling, data filtering
-   **Skills**: Reactive programming, UI/UX design, server-side logic
-   **Output**: Complete web application with user interface

## 🛠️ Technologies Mastered

### Core R Packages

``` r
# Interactive Visualization
library(shiny)      # Web application framework
library(leaflet)    # Interactive mapping
library(plotly)     # Interactive plotting
library(DT)         # Interactive tables

# Web Development  
library(htmlwidgets) # HTML widget creation
library(bslib)      # Bootstrap themes
library(shinyWidgets) # Enhanced UI components

# Data Processing
library(dplyr)      # Data manipulation
library(ggplot2)    # Base plotting for plotly
```
